# ubuntuInstall
Setup Script for Ubuntu Machines

The script will install the following software:
* Android
* Apache 
* Bower
* Composer
* Cordova
* Curl
* Git
* Grunt
* Ionic
* Java 8 (Oracle)
* Mongodb
* MySQL
* Nodejs / NPM
* PHP
* phpmyadmin
* Python (Pip, Virtualenv)
* Sublime
* Ubuntu Make
* VirtualStudio Code

Will also attempt to install the android sdk packages for:
* SDK v22
* SDK v19
* build tools
* platform tools
